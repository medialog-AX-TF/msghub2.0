## 2. 도구 정의 및 시스템 프롬프트 설계 관련 프롬프트

### 2.1 도구(Tool) 정의 및 스키마 작성

```
메시지허브 POC를 위한 웹페이지 자동화 도구를 JSON 스키마 형식으로 정의해주세요. 다음 도구들이 필요합니다:

1. navigate_to: 특정 메뉴 또는 URL로 이동
   - 매개변수: menu_name 또는 url
   - 반환: 이동 성공 여부

2. click_element: 버튼, 링크, 체크박스 등 요소 클릭
   - 매개변수: element_id 또는 element_selector
   - 반환: 클릭 성공 여부

3. input_text: 텍스트 필드에 내용 입력
   - 매개변수: element_id, text
   - 반환: 입력 성공 여부

4. extract_data: 웹 페이지에서 데이터 추출
   - 매개변수: selector 또는 xpath
   - 반환: 추출된 데이터

각 도구에 대해 JSON 스키마 형식으로 정의하고, 필수 및 선택적 매개변수, 반환 값 형식, 사용 예시를 포함해주세요.
```
